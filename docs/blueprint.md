# **App Name**: Student Birthday Reminder

## Core Features:

- Admin Account Authentication: Secure login for the professor using email and password, restricted to a single administrator account with no public registration.
- Student Profile Management: Professor can add new student details, edit existing entries, and delete student records. Each student profile includes Name, Roll Number, Department, Birthday Date, and an optional photo.
- Birthday Overview Dashboard: A dashboard display highlighting students celebrating a birthday on the current day with a dedicated card, alongside an in-app popup notification system for daily birthday alerts.
- Interactive Birthday Calendar: A visually intuitive calendar interface displaying all student birthdays, allowing the professor to easily view upcoming celebrations.
- Searchable Student Directory: A comprehensive list of all enrolled students, presented in a table format with search, filter, and sort capabilities to easily locate student information.

## Style Guidelines:

- The app uses a light color scheme designed for a soft and welcoming aesthetic. The primary color is a gentle rose (#D689B9) to convey warmth and care. The background is a very subtle pale pink (#FAE9F5) that offers a clean, airy feel. A rich, deeper plum (#9D38DB) serves as the accent color, providing contrast for interactive elements and highlights.
- The chosen typeface is 'Inter' (sans-serif) for all text elements, providing a modern, objective, and highly legible appearance suitable for both headlines and body text on a clean dashboard.
- Employ a set of clean, minimalist line icons throughout the interface, consistently applied for navigation, action buttons (e.g., edit, delete), and feature representations (e.g., calendar, notification bell). Consider a celebratory icon like a birthday cake for relevant features.
- A modern, card-based layout characterizes the dashboard, ensuring information is well-organized and easily digestible. The design emphasizes responsiveness, adapting seamlessly to both mobile and desktop screens for an optimal user experience across devices.
- Implement a 'confetti' animation that triggers specifically on student birthdays shown on the dashboard, adding a festive touch. Utilize subtle transition effects and hover states for interactive UI elements to enhance user engagement and provide visual feedback.
